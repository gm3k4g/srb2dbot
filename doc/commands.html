srb2dbot v0.1 Overview Commands Building Architecture Tools

Slash Commands

All commands require the p_ban_members Discord permission. Commands are registered as guild-specific slash commands.

Script Editing Commands

These commands edit ~/.srb2/srb2_servers.d/srb2b.d/srb2b.sh - the bash configuration script that runs the SRB2 server. Password lines are redacted in output.

/get_script

Displays the entire server configuration script (with password lines redacted).

/find_line <target_string>

Searches the script for lines containing the target string (case-sensitive). Returns line numbers and contents of matches.

Parameters: target_string - string to search for

/inspect_line <line_num>

Shows a range of lines around the target line (approximately 8 lines above and below), with the target line highlighted with an arrow (->).

Parameters: line_num - the line to inspect (1-based)

/insert_line <line_num> <line_contents>

Inserts a new line of text at the specified position, shifting existing lines down. Empty content inserts a blank line.

Parameters: line_num - position to insert at; line_contents - text to insert

/remove_line <line_num>

Removes the specified line from the script. Shifts subsequent lines up.

Parameters: line_num - the line to remove (1-based)

/change_line <line_num> <line_contents>

Replaces the content of the specified line with new text.

Parameters: line_num - line to change; line_contents - replacement text

/move_line <old_line_num> <new_line_num>

Moves a line from one position to another. The line at the destination shifts to accommodate. When moving downward, the destination line stays before the moved line. When moving upward, the moved line goes before the destination.

Parameters: old_line_num - source position; new_line_num - target position

WAD Management Commands

These commands operate on ~/.srb2/addons/.

/list_wads

Lists all WAD/PK3 files currently in the addons directory.

/search_wads <target_filename>

Searches for WAD files whose names contain the target string (case-insensitive).

Parameters: target_filename - string to match against filenames

/addfile_upload <file>

Uploads a file attached to the Discord message to the server's addons directory. Files are capped at 100 MB. Filenames are sanitized to prevent path traversal.

Parameters: file - Discord file attachment

/addfile_link <file_url>

Downloads a file from a URL and stores it in the addons directory. The filename is extracted from the URL path. Files are capped at 100 MB.

Parameters: file_url - direct download URL

Server Control Commands

/restart_server

Restarts the SRB2 server via systemd. Equivalent to systemctl restart <service> --user. The service name defaults to srb2@srb2b and can be changed in secret.json.

/stop_server

Stops the SRB2 server via systemd. Equivalent to systemctl stop <service> --user.

/server_do <server_command>

Sends a raw command to the SRB2 server console via the FIFO pipe. Input is sanitized - control characters and newlines are rejected.

Parameters: server_command - console command to execute

Player Management Commands

/server_say <server_message>

Sends a chat message to the server visible to all players.

Parameters: server_message - text to broadcast in-game

/kick_player <player>

Kicks a player from the server by name or node number.

Parameters: player - player name or node number to kick

/ban_player <player>

Bans a player from the server by name or node number.

Parameters: player - player name or node number to ban
