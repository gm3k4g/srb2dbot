Companion Tools

These bash scripts assist with media generation and testing. All are located in the project root and require ImageMagick (magick or convert).

Intermission Overlay Generator

./generate_intermission.sh - creates a transparent PNG overlay that mimics SRB2's intermission results screen.

Usage

# FFA mode (single column, sorted by score)
./generate_intermission.sh \
    --gametype ffa \
    --map "MAP01" \
    --title "Green Flower Zone" \
    --players '[
        {"name":"Sonic","score":4500},
        {"name":"Tails","score":3200},
        {"name":"Knuckles","score":2800}
    ]' \
    --out intermission_ffa.png

# Team mode (blue/red columns with team scores)
./generate_intermission.sh \
    --gametype team \
    --map "MAP01" \
    --title "Green Flower Zone" \
    --blue-score 3 \
    --red-score 5 \
    --players '[
        {"name":"Sonic",  "score":4500, "team":"blue"},
        {"name":"Knuckles","score":2800, "team":"blue"},
        {"name":"Tails",  "score":3200, "team":"red"},
        {"name":"Amy",    "score":1500, "team":"red"}
    ]' \
    --thumb "~/.srb2/luafiles/client/DiscordBot/thumbnails/MAP01.png" \
    --out intermission_team.png

Player JSON Format

[
    {"name":"Player1","score":1500,"team":"blue"},
    {"name":"Player2","score":1200,"team":"red"}
]

The team field is only used in team mode. Accepted values: "blue" or "red". Omit or set to null for FFA mode. Players are sorted by score descending within each team.

Title Bar Format

The header bar shows:

LineAlignContentFont sizeExample
Top (y=22)CenterGametype22pxFFA
Top (y=22)RightRound time22px3:45
Bottom (y=44)CenterMap name (code)16pxGreen Flower Zone (MAP01)

Gametype displays as FFA or Team. Omitting a field excludes it.

Clipping Warning

If more players are provided than fit in the panel area, a warning is printed to stderr:

WARNING: Only 9 of 12 players fit in panel (3 clipped)
WARNING: Only 20 of 24 players fit (4 clipped)

FFA mode fits up to 9 rows, team mode fits up to 10 rows per column (20 total).

Output

  • Transparent PNG overlay with alpha channel
  • If --thumb is provided, the thumbnail is resized to fill the canvas and the overlay is composited on top
  • If no thumbnail is available, only the transparent overlay is output (suitable for manual compositing)

Thumbnail Grabber

./generate_thumbnails.sh - extracts SRB2 level select pictures (MAPxxP.lmp) from PK3 files and converts them to PNG.

Usage

# Scan default SRB2 directory
./generate_thumbnails.sh

# Specify custom maps and output directories
./generate_thumbnails.sh /path/to/pk3/files /path/to/output

# Extract PK3s first
find /path/to/addons -name '*.pk3' -exec unzip -o {} -d {}_FILES \
    'Level select pictures/*' \;

Hitbox Validation

The script validates rectangle positions before generating the image. All panel backgrounds, header highlights, and row backgrounds are tracked as hitboxes. If any two non-nested rectangles overlap (share interior area), the script exits with an error showing the overlapping coordinates and labels. Nested containment (e.g. a row background inside a panel) is permitted.

Spectators

Optional spectator names are displayed at the bottom of the panel in small (11px), semi-transparent text.

./generate_intermission.sh ... \
    --spectators '["Guest1","Guest2","Viewer3"]'

# Or from a file
./generate_intermission.sh ... \
    --spectators-file spectators.json

Spectator JSON format: ["name1", "name2", ...]

Intermission Test Harness

./test_intermission.sh - generates sample outputs for 2, 4, 8, 12, 16, 18, and 24 players in both FFA and Team modes, verifying that all hitboxes pass validation and outputs are valid transparent PNGs.

./test_intermission.sh [output_directory]
# Default: /tmp/intermission_tests/

Dependencies

  • ImageMagick (for both tools)
  • unzip (for PK3 extraction, thumbnail grabber only)
